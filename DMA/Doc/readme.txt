【*】 程序简介 
-工程名称：DMA―存储器到存储器模式
-实验平台: 野火STM32 MINI 开发板
-MDK版本：5.16
-ST固件库版本：3.5.0


【 ！】功能简介：
使用DMA传输将源数据拷贝到指定地址存储空间上，并比较源数据与传输后数据是否相等。

学习目的：学习DMA传输使用，特别是存储器到存储器传输模式功能实现方法。

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配
LED：
LED灯的两个阴极接到STM32两个个引脚上，LED低电平亮。

	LED D4<--->PC2
	LED D5<--->PC3

/*********************************************************************************************/

【*】程序描述：

<main.c>

	1.使能DMA时钟
	2.配置DMA数据参数
	3.使能DMA,进行传输
	4.等待传输完成，并对源数据和目标地址数据进行比较。

【*】注意：当使用存储器到存储器模式时候，通道（CHANNEL)可以随便选，没有硬性的规定，而存储器到外设和外设到存储器不可以
